안드로이드 WiFi ADB 방법


안드로이드 OTG개발중 USB 테스트에 불편함으로 찾음



## Install

1. Prefrences -> Plugins -> Browse Repositories 이동

2. 검색에 Android wifi adb 검색

3. 안드로이드 스튜디오 재시작

4. USB 연결후 Android WiFi ADB 버튼으로 연결후 사용
