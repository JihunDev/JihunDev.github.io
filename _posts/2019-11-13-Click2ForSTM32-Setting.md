Click2 For STM32 Setting

## 개발환경
- MikroC Pro for ARM v.6.0.0
- MikroProg v.1.60
- MCU : Click 2 For STM32

------------
## Setting
### MikroProg
1. 프로그램 Admin Open
2. Options And set Connection : Normal

### MikroC Pro for ARM
1. Compiler go to project
2. Edit project
3. Load Scheme
4. "STM32F407VG_Int_OSC_16MHZ.cfsch" 맞는 파일 선택
